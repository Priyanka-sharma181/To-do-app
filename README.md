# Node.js TODO App

This is a simple TODO app built with Node.js. It allows you to manage your tasks efficiently.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- List all tasks

## Prerequisites

Make sure you have the following installed on your machine:

- Node.js
- npm (Node Package Manager)
